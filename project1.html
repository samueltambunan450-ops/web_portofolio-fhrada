Project 01

Website Kuliner Khas Medan

Project ini menjadi salah satu pondasi awal perjalanan saya di web development. Fokus utamanya adalah menyajikan informasi kuliner khas Medan dalam tampilan yang mudah dipahami, menarik, dan nyaman dijelajahi.

HTML CSS JavaScript PHP

Ringkasan project

Website ini dibuat untuk menampilkan berbagai rekomendasi kuliner khas Medan dengan pendekatan yang informatif. Dari project ini saya belajar menyusun halaman, membangun navigasi yang jelas, dan merancang tampilan yang membuat pengguna lebih mudah menemukan informasi yang mereka butuhkan.

Hal yang saya pelajari

  • Menyusun struktur halaman web yang rapi dan mudah dikembangkan.
  • Menggabungkan visual, konten, dan navigasi agar pengalaman pengguna lebih jelas.
  • Membangun fondasi project yang kemudian bisa dikembangkan ke versi berikutnya.

Project ini menjadi titik awal saya dalam memahami bagaimana sebuah website dibangun dengan lebih terstruktur.
