Project 01

Medan Culinary Website

This project became one of the early foundations of my journey in web development. Its main focus was presenting Medan culinary information in a way that feels clear, attractive, and easy to explore.

HTML CSS JavaScript PHP

Project summary

This website was created to showcase a variety of Medan culinary recommendations through an informative approach. From this project, I learned how to structure pages, build clear navigation, and design interfaces that help users find the information they need more easily.

What I learned

  • How to build a clean web page structure that is easy to develop further.
  • How to combine visuals, content, and navigation for a clearer user experience.
  • How to create a project foundation that can grow into the next version.

This project became my starting point in understanding how a website can be built in a more structured way.
