Project 02

Rekomendasi Kuliner Khas Medan

Project ini merupakan pengembangan lanjutan dari website kuliner yang saya kerjakan sebelumnya. Di tahap ini saya mulai masuk ke alur aplikasi yang lebih tertata dengan memanfaatkan Laravel agar struktur project terasa lebih rapi dan siap berkembang.

Laravel PHP MySQL CRUD Flow

Ringkasan project

Website ini saya kembangkan untuk memberikan rekomendasi kuliner yang lebih terstruktur. Penggunaan Laravel membantu saya memahami pola pengembangan yang lebih baik dibanding project awal, terutama dalam pengelolaan halaman, data, dan alur penggunaan aplikasi.

Hal yang saya pelajari

  • Mengenal workflow Laravel untuk pengembangan website yang lebih terorganisir.
  • Memahami pemisahan struktur tampilan, logika, dan pengelolaan data dengan lebih baik.
  • Mengembangkan project lama menjadi versi yang terasa lebih matang secara teknis.

Project ini memperkuat pemahaman saya tentang transisi dari website dasar ke framework yang lebih siap digunakan.
