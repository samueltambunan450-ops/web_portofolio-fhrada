Project 03

WorkRadar

WorkRadar adalah project kolaboratif berbasis mobile app. Pada project ini saya berkontribusi di bagian fitur cuaca dan lokasi, sehingga pengguna bisa mengetahui kondisi sekitar mereka dengan lebih praktis.

Flutter Open-Meteo API Geolocator Team Project

Ringkasan project

Project ini memperluas pengalaman saya dari web ke aplikasi mobile. Saya belajar bagaimana fitur berbasis data real-time seperti cuaca dan lokasi membutuhkan integrasi API, pengambilan posisi pengguna, dan koordinasi yang baik di dalam tim.

Hal yang saya pelajari

  • Mengintegrasikan data cuaca dari Open-Meteo API ke dalam alur aplikasi.
  • Menggunakan geolokasi untuk membantu aplikasi membaca konteks lokasi pengguna.
  • Berkolaborasi sebagai anggota tim pada project dengan ruang lingkup yang lebih besar.

Project ini membantu saya memahami pentingnya kolaborasi dan integrasi layanan eksternal pada aplikasi modern.
