WorkRadar

WorkRadar is a collaborative mobile app project. In this project, I contributed to the weather and location features so users could check their surrounding conditions more easily.

Flutter Open-Meteo API Geolocator Team Project

Project summary

This project expanded my experience from web into mobile apps. I learned how real-time data features such as weather and location require API integration, user position handling, and strong team coordination.

What I learned

  • Integrating weather data from the Open-Meteo API into the application flow.
  • Using geolocation to help the app understand the user's location context.
  • Collaborating as a team member on a project with a broader scope.

This project helped me understand the importance of collaboration and external service integration in modern applications.
